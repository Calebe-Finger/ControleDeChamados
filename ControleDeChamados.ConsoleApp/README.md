# Gerenciamento de Chamados

![](.gif)

## Introdução 

Trata-se de um programa criado para a gestão de chamados de 
forma rápida e fácil.

## Funcionaçidades

- O usuário pode criar, editar visualizar e excluir chamados.
- O ID é fornecido automáticamente ao usuário, não necessitando
a inserção do mesmo de forma manual.
- O Programa foi criado seguindo o método de 
Orientação a Objetos, facilitando a compreenção, e eventuais
mudanças no mesmo.
